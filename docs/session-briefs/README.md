# Session Briefs Archive

This folder stores strategy briefs dragged from Claude.ai into Claude Code sessions. This creates a permanent record of strategic decisions — Claude.ai conversations are ephemeral, the repo is the record.

## Naming Convention

`session-[N]-[topic].md` — e.g., `session-15-twitter-strategy.md`

## Rule

When John drags a brief from Claude.ai into Claude Code, Claude Code must save it here before implementing. The brief is the audit trail.
